## About `inhibicion`

`inhibicion` comprises the PhD work of [@felipeyanez](https://github.com/felipeyanez) characterizing the diversity of cortical inhibitory neurons.

- [**decoupling**](https://github.com/inhibicion/decoupling) is the main repository describing how inhibitory neuron diversity can be decoupled.
- [**column**](https://github.com/inhibicion/column) is a dataset repository containing the morphology and electrophysiology of a cortical column.
- [**moma**](https://github.com/inhibicion/moma) is a dataset repository quantifying the molecular composition of the barrel cortex.

See [Dissertation](http://dx.doi.org/10.15496/publikation-104703) or [Public Lecture](https://www.youtube.com/watch?v=yK4xWW_3z9E) for further details.
